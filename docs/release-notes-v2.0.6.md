# OpenClaw Chat Gateway v2.0.6

## 重点改进

- “关于系统”里的升级入口改为更清晰的阶段式交互：
  - 支持按升级阶段显示普通用户可读文案
  - 升级成功后直接显示“点击重启服务”
  - 升级失败时可直接展示失败原因
- 升级进行中点击停止时，改为先弹项目内确认弹窗，再决定是否调用停止接口，不再直接中断。

## 界面优化

- “检查新版本”按钮恢复为与原有设置页一致的紧凑样式：
  - 默认状态按文案自适应宽度
  - 长文案状态只横向变长，不再被撑高
  - 边框、背景色、悬停态与现有蓝色按钮样式保持一致

## 多语言

- 本次关于页升级交互新增文案已同步补齐 `zh-CN`、`zh-TW`、`en` 三语资源。
