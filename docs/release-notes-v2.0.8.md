# OpenClaw Chat Gateway v2.0.8

这次更新重点是把 OpenClaw 的更新和连接状态做得更清楚、更可靠。

## 这次有什么变化

- OpenClaw 更新过程现在会按阶段显示：
  - 正在下载更新包
  - 正在安装更新包
  - 正在修复命令入口
  - 正在校验升级后的版本
- 可安全停止的更新阶段会明确提示“点击停止”，避免看起来像卡住。
- OpenClaw 已经恢复后，页面里的连接状态会更快跟上，不再长时间误报“未连接”。
- 网关状态检测和“测试连接”判断更快、更准，优先按本机 OpenClaw 实际运行态返回结果。

## 适合谁升级

- 如果你在页面里更新 OpenClaw，希望能更清楚看到更新进度
- 如果你遇到过 OpenClaw 已经起来了，但页面还要等很久才恢复连接状态
